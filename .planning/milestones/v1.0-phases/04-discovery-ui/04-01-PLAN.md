   - `getWeekendSagre()`: Fetch active sagre where dates overlap with today through +3 days. Use the query pattern: `.eq("is_active", true)` then `.or(`end_date.gte.${today},end_date.is.null`)` combined with `.lte("start_date", threeDaysOut)`. Select only SAGRA_CARD_FIELDS columns. Order by start_date ASC. Limit 8. Return `SagraCardData[]`.
   - `searchSagre(filters: SearchFilters)`: If lat+lng provided, call `supabase.rpc("find_nearby_sagre", { user_lat, user_lng, radius_meters: (raggio ?? 30) * 1000, max_results: 50 })` then apply additional filters (provincia, cucina, gratis, date range) in-memory on the results. Order by start_date ASC, limit 50. Return `SagraCardData[]`.
   - `getProvinceCounts()`: Call `supabase.rpc("count_sagre_by_province")`. Return `ProvinceCount[]`.

Important notes:

- The weekend query must catch multi-day sagre whose start_date is in the past but end_date is in the future (or null, treated as single-day).
