       - Props: `{ lat: number; lng: number }`
       - Helper function: `getDirectionsUrl(lat, lng) => https://www.google.com/maps/dir/?api=1&destination=${lat},${lng}`
       - Render: `Button` with `variant="outline"` wrapping an `<a>` tag with `target="_blank"` and `rel="noopener noreferrer"`

       - Display: `Navigation` icon (lucide-react) + text "Indicazioni"
       - The `<a>` tag approach is better than Button+onClick for accessibility and works without JS

    2. Create `src/components/detail/ShareButton.tsx`:
       - Mark `"use client"` (needs useState and browser APIs)
       - Import `useState` from `react`
       - Import `Share2`, `Check` from `lucide-react`
       - Import `Button` from `@/components/ui/button`
       - State: `copied` boolean, initially false
       - `handleShare` async function:
         - Try `navigator.clipboard.writeText(window.location.href)`, set `copied = true`, setTimeout 2s to reset
         - Catch: fallback to `navigator.share({ url: window.location.href })` if available on mobile
       - Render: Button with `variant="outline"`, shows "Copiato!" with Check icon when copied, "Condividi" with Share2 icon when not copied

    3. Create `src/components/detail/SagraDetail.tsx`:
       - Server component (no "use client")
       - Props: `{ sagra: Sagra }` imported from `@/types/database`
       - Import `Image` from `next/image`, icons from `lucide-react` (MapPin, Calendar, Tag, Euro, ExternalLink, Clock), `Badge` from ui, `formatDateRange` from utils
       - Import `DirectionsButton` and `ShareButton`
       - Import `DetailMiniMapDynamic` from `./DetailMiniMap.dynamic`
       - Layout (mobile-first, single column):
         a. **Hero image** (if `image_url`): `relative h-48 w-full -mx-4 -mt-4` to break out of padding, Image with `fill` and `object-cover`
         b. **Title**: `text-2xl font-bold` with the sagra title
         c. **Location row**: MapPin icon + `location_text` + province in parentheses
         d. **Date row**: Calendar icon + `formatDateRange(start_date, end_date)`
         e. **Price row** (if `price_info` or `is_free`): Euro icon + either "Ingresso gratuito" (if `is_free`) or `price_info`
         f. **Tags section** (if `food_tags` or `feature_tags`): Tag icon + badges for each tag
         g. **Description section** (if `enhanced_description` or `description`): heading "Descrizione" + paragraph text. Prefer `enhanced_description`, fall back to `description`.
         h. **Mini map** (if `location` is not null): heading "Posizione" + `DetailMiniMapDynamic` with lat/lng extracted from `location.coordinates` (remember: [lng, lat] from PostGIS)
         i. **Action buttons row**: `DirectionsButton` (if location exists) + `ShareButton` side by side in a flex row
         j. **Source link** (if `source_url`): ExternalLink icon + `<a>` tag to `source_url` with text "Vedi sito originale", `target="_blank"`, `rel="noopener noreferrer"`, styled as muted link

       - Use `space-y-6` for section spacing, consistent icon sizes (`h-4 w-4`)
       - Use Tailwind classes matching the existing design system (stone bg, amber primary, etc.)
