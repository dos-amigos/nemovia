**Step 1: Create FadeImage component (MICRO-05)**

Create `src/components/animations/FadeImage.tsx` as a client component:
```tsx
"use client";

import { useState, useRef, useEffect } from "react";
import Image, { type ImageProps } from "next/image";
import { cn } from "@/lib/utils";

export function FadeImage({ className, ...props }: ImageProps) {
  const [loaded, setLoaded] = useState(false);
  const imgRef = useRef<HTMLImageElement>(null);

  // Handle cached images that may not fire onLoad
  useEffect(() => {
    if (imgRef.current?.complete) {
      setLoaded(true);
    }
  }, []);

  return (
    <Image
      {...props}
      ref={imgRef}
      className={cn(
        "transition-opacity duration-500 ease-in-out",
        loaded ? "opacity-100" : "opacity-0",
        className
      )}
      onLoad={() => setLoaded(true)}
    />
  );
}
```
The `useEffect` check for `img.complete` handles cached images where the browser may not fire the `onLoad` event (Pitfall 6 from RESEARCH.md).

**Step 2: Convert SagraCard to client component with motion gestures (MICRO-01, MICRO-02, TRANS-02)**

Modify `src/components/sagra/SagraCard.tsx`:
1. Add `"use client";` directive at the top. This is safe because SagraCard's parent (StaggerGrid) is already a client component, so this changes nothing in the rendering tree (Pitfall 2 from RESEARCH.md).
2. Add import: `import { motion } from "motion/react";`
3. Replace `import Image from "next/image"` with `import { FadeImage } from "@/components/animations/FadeImage";`
4. Inside the Link, wrap the Card in a motion.div:
```tsx
<Link href={`/sagra/${sagra.slug}`} className="block rounded-lg focus-visible:ring-[3px] focus-visible:ring-ring/50">
  <motion.div
    whileHover={{ scale: 1.02, boxShadow: "0 8px 30px rgba(0,0,0,0.12)" }}
    whileTap={{ scale: 0.97 }}
    exit={{ scale: 1.05, opacity: 0, transition: { duration: 0.15 } }}
    transition={{ type: "spring", stiffness: 400, damping: 25 }}
  >
    <Card className="overflow-hidden py-0">
      {/* existing card content */}
    </Card>
  </motion.div>
</Link>
```
- `whileHover` fires only on pointer devices (desktop) = MICRO-01 card lift
- `whileTap` fires on both touch and mouse = MICRO-02 tap feedback
- `exit` creates a brief scale-up + fade = TRANS-02 simulated morph (card expands before fading out during page transition)
5. Replace the `<Image>` tag (where `sagra.image_url` is rendered) with `<FadeImage>`, keeping all existing props (src, alt, fill, className, sizes).
6. Remove the `hover:shadow-md transition-shadow` classes from the Card since the motion.div now handles the shadow via `whileHover.boxShadow`.
