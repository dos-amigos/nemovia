Create `usePreviousValue<T>(value: T)` hook that stores previous value via useRef.

- Gets previous context via usePreviousValue
- Gets current and previous segment via useSelectedLayoutSegment + usePreviousValue

- This prevents the new route from rendering during the exit animation

- Gets current segment via useSelectedLayoutSegment

  - `initial={{ opacity: 0 }}`
  - `animate={{ opacity: 1, transition: { duration: 0.15, ease: "easeOut" } }}`
  - `exit={{ opacity: 0, transition: { duration: 0.1 } }}`
- Children wrapped in FrozenRouter

Keep durations short (150ms enter, 100ms exit) per the Out of Scope mandate against complex enter/exit animations that hurt utility app speed.
