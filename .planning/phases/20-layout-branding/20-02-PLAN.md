**1. Create `src/components/brand/Logo.tsx`:**

Create an inline SVG React component for the Nemovia logo. The logo should be a wordmark with a small icon element. Design specifications:

- SVG viewBox sized for navbar height (target h-7 to h-8, ~28-32px tall)
- A small geometric icon on the left: a stylized fork/map-pin hybrid or a simple circle/diamond mark using `var(--accent)` (teal)
- The word "Nemovia" to the right of the icon in Geist Bold style weight, using `var(--primary)` (coral) for color
- Use `fill="var(--primary)"` for text elements and `fill="var(--accent)"` for the icon mark -- CSS custom properties work because the SVG is inline
- Add `aria-label="Nemovia"` and `role="img"` on the SVG element for accessibility
- Accept a `className` prop for external sizing (e.g., `h-7 w-auto`)
- Export as named export: `export function Logo({ className }: LogoProps)`
- The SVG text should use a clean geometric sans-serif style. Since we cannot embed the Geist font in SVG easily, use `<path>` elements for the wordmark letters (not `<text>`). This ensures the logo renders identically everywhere. Use a clean, modern geometric letterform style.
- Keep the SVG clean and minimal -- roughly 140x32 viewBox

The icon element should be something simple and recognizable: a small circle with a fork silhouette, or a location pin with a plate/utensil motif, or even a simple abstract mark. Keep it to 2-3 SVG paths maximum. The icon uses teal (`var(--accent)`) and the wordmark uses coral (`var(--primary)`).
